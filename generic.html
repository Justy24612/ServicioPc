Hack PlayStation 3

información:
-Se instala un hack en la consola, la cual permite ejecutar copias de seguridad (juegos) de manera gratuita
-El hack no permite jugar de manera online solo de manera local
-Se instalara una tienda que permite al usuario descargar juegos sin costo alguno
-El usuario podrá instalar juegos desde un pendrive siempre y cuando este sea compatible con la consola
-Con el pago de este servicio se instalaran 2 juegos de la lista sin costo a pedido del usuario
-Se debe activar el hack cada vez que se prende la consola. este no tiene dificultad solo toma unos segundos en activarse
